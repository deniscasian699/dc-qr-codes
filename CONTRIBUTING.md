# Contributing to DC QR CODES

Thank you for your interest in contributing to DC QR CODES! 🎉

While the source code is proprietary, there are several ways you can help improve the project.

## 🐛 Bug Reports

Found a bug? Please help us fix it!

**Report bugs at:** [bugs.deniscasian.com](https://bugs.deniscasian.com)

Please include:
- Clear, descriptive title
- Steps to reproduce
- Expected vs actual behavior
- Browser and device info
- Screenshots if applicable

## 💡 Feature Requests

Have an idea to make DC QR CODES better?

**Submit feature requests at:** [qa.deniscasian.com](https://qa.deniscasian.com)

Please describe:
- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

## 🌐 Translations

Help us reach more users by adding translations!

Currently supported:
- 🇬🇧 English
- 🇷🇴 Romanian

To add a new language:
1. Contact us at [support@deniscasian.com](mailto:support@deniscasian.com)
2. We'll provide the translation file template
3. Submit your translations
4. Get credited in the app!

## ⭐ Show Support

The easiest way to contribute:

- **Star** this repository
- **Share** with friends and colleagues
- **Get a [Pro License](https://digitalstore.deniscasian.com/l/dc-qr-codes-pro)** to support development
- **Join our [Discord](https://discord.com/invite/M3RaXJpenV)**

## 📜 Code of Conduct

Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before participating.

## 📋 Guidelines

- Be respectful and constructive
- Provide as much detail as possible
- Use English for all communications
