## ✨ Features

### 🎨 Full Customization
- Custom QR code colors
- Custom background colors
- Adjustable size (128px - 512px)
- Optional center logo
- 4 error correction levels

### 📥 Export Options
- Download as PNG (raster)
- Download as SVG (vector)
- High-quality output
- Perfect for print & digital

### 💾 Local Storage
- QR history saved locally
- No account required
- Full privacy - data never leaves your device
- Quick access to recent codes

### 🌐 Multi-Language
- English 🇬🇧
- Romanian 🇷🇴
- Easy language switching
- Auto-detect browser language

### 📱 PWA Support
- Install as desktop/mobile app
- Works offline
- Fast loading
- Native app experience

### 🔒 Privacy First
- No tracking
- No data collection
- Everything runs in browser
- Open source code

## 🔒 Privacy & Security

DC QR CODES is designed with privacy as a core principle:

- **🏠 Local Processing**: All QR codes are generated in your browser
- **🚫 No Tracking**: We don't track your usage or generated codes
- **💾 Local Storage**: History is stored only in your browser's localStorage
- **🔐 No Account**: No registration or login required
- **📡 Minimal Network**: Only language files and optional license verification
